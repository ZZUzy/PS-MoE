# PS-MoE
A reliability-aware pairwise learning framework for DTI prediction with interaction-driven graph representations

This repository contains the source code and the data.

PS-MoE is a reliability-aware framework for drug–target interaction prediction. It models explicit atom–residue interaction pathways and adaptively fuses sequence- and graph-based experts using reliability supervision. It improves generalization and supports interpretable candidate prioritization.
